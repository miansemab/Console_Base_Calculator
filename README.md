# 🧮 Simple Calculator

A clean and beginner-friendly **Simple Calculator** project that performs basic arithmetic operations. This project is perfect for learning fundamental programming concepts such as user input, condition handling, and functions.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧠 Easy-to-understand logic
* 🔁 Continuous calculations (optional loop support)
* ⚠️ Input validation (optional advanced version)

---

## 🛠️ Technologies Used

* Python (or update with your language)
* Console-based interface
