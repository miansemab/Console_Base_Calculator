# 🧮 Console Base Calculator

A simple and efficient **console-based calculator** built in Python that supports basic arithmetic operations. This project is perfect for beginners who want to understand how calculations work behind the scenes using programming logic.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🔁 Continuous calculations (multiple operations in one run)
* ⚠️ Input validation to prevent errors
* 💡 Beginner-friendly and easy to understand

---

## 🛠️ Technologies Used

* **Python 3**
* Standard Python libraries (no external dependencies)

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/miansemab/Console_Base_Calculator.git
```

2. Navigate to the project folder:

```bash
cd Console_Base_Calculator
```

3. Run the program:

```bash
python calculator.py
```
